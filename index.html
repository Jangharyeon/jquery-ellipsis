jQuery Dropdown

Selector

  • Button: data-dropdown="button"
  • Panel: data-dropdown="panel"

Options

  • animateSpeed
    • Type: Number
    • Default: 150
  • fade
    • Type: Boolean
    • Default: false
  • slide
    • Type: Boolean
    • Default: false
